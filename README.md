# Vibe Plan Space

Vibe Plan Space is a beautifully designed, modern productivity and planning application. It features a stunning glassmorphism interface and helps users stay organized with habit tracking, task management, mood logging, and seamless cloud synchronization.

## Features

- **Habit Tracking:** Build and maintain daily habits with a visual 7-day streak tracker.
- **Task Management:** Organize tasks effectively using the Eisenhower Matrix (Urgent vs Important).
- **Secure Cloud Sync:** Real-time data backup and cross-device synchronization using Firebase Authentication and Firestore.
- **Modern UI:** Vibrant aesthetics, smooth micro-animations, and responsive design built with Tailwind CSS.

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AvneetSingh11/vibe-plan-space.git
   ```
2. Navigate to the project directory:
   ```bash
   cd vibe-plan-space
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server, run:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal) to view the application.

### Building for Production

To create a production build, run:
```bash
npm run build
```

## Technologies Used

- **React** - Frontend library
- **Vite** - Next Generation Frontend Tooling
- **Tailwind CSS** - Utility-first CSS framework
- **Firebase** - Authentication and Cloud Firestore Database
